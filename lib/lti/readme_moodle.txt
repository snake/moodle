Installation
------------

1) Download the latest version of the provider library
wget https://github.com/snake/lti-1-3-php-library/archive/refs/tags/v1.0.0.zip

2) Unzip the archive
unzip v1.0.0.zip

3) Move the source code directory into place
mv lti-1.3-library-php/src/lti/* lib/lti/

4) Move the LICENSE and README.md into place
mv lti-1.3-library-php/LICENSE lib/lti/
mv lti-1.3-library-php/README.md lib/lti/

5) Run unit tests on enrol_lti_testsuite
